| s13 | 색상전체제거 | 놓은 공과 같은 색상 보드에서 모두 제거 (점수 있음) | 0.25% | c6 500개 모으기 |
| s14 | 보드전체제거 | 보드 모든 타일 제거 (점수 있음) | 0.25% | c7 500개 모으기 |

### 주변(8방향) 정의
놓은 위치 기준 상하좌우 + 대각선 4곳. 보드 밖은 제외.
- 모서리: 3곳, 변: 5곳, 가운데: 8곳

### 스킬 발동 순서
1. 스포이드(s12)만 놓기 전에 발동 (2단계: 보드 공 클릭 → 색 복사 → 배치)
2. 나머지 스킬은 공 배치 후, 상쇄(매칭) 전에 발동
3. "상쇄 먼저"로 표기된 스킬: 스킬 효과 적용 → 바로 매칭 체크

## 3. 해금 시스템

### 타일 타입 해금 (판수 기반, 영구)
| 판수 | 해금 |
|------|------|
| 0 | type 3만 (색상지정 줄공) |
| 100 | type 4 (색상미지정 줄공) |
| 200 | type 5 (색상지정 십자공) |
| 300 | type 6 (색상미지정 십자공) |
| 500 | type 1 (색상지정 자유배치) |
| 1000 | type 2 (색상미지정 자유배치) |

### 색상 해금 (점수 기반, 매 게임)
| 점수 | 해금 |
|------|------|
| 0 | 4색 (c1~c4) |
| 500 | +c5 |
| 1000 | +c6 |
| 1500 | 자동 색변경 |
| 2000 | +c7 |
| 2500 | 무효블록 생성 |

### 스킬 해금 조건 (영구)
- 보드클리어 → s1
- 3콤보 → s2
- 4콤보 → s3
- 5콤보 → s4
- c5 100개 → s5, c6 100개 → s6, c7 100개 → s7
- c5 300개 → s8, c6 300개 → s9, c7 300개 → s10, s11
- c5 500개 → s12, c6 500개 → s13, c7 500개 → s14

### 미션 연속 성공 보너스
- 연속 2회: 미션점수 +10
- 연속 3회: 미션점수 x2 (+10 안함)

## 5. UI 변경사항

### 큐 프리뷰 공
- 스킬 있는 공: 오른쪽 위에 작은 스킬 아이콘/뱃지 표시
- 스킬 아이콘은 작은 원형 or 심볼

### 게임오버 화면
- 기존: 최종 점수, 최대 콤보
- 추가: "해금!" 목록 (이번 게임에서 새로 해금된 것들)
- 추가: 수집 현황 (c5/c6/c7 몇 개)

### 스포이드 모드
- s12 스킬 공이 Current일 때
- 상태: "스포이드: 색을 복사할 공을 선택하세요"
- 보드의 공 클릭 → 색 복사 → 일반 배치 모드로 전환

### s4, s7 선택 모드
- 공 배치 후 "변경/제거할 주변 타일을 선택하세요"
- 주변 8방향 타일 하이라이트
- 클릭하면 효과 적용

## 7. 주의사항
- "상쇄 먼저" 스킬: 스킬 효과 → 매칭 체크 → 콤보/미션 판정
- 스포이드는 유일하게 배치 전 발동
- 스킬 확률은 해금된 스킬만 대상 (미해금 스킬은 확률 풀에 안 들어감)
- 스킬 없는 공 확률 = 100% - (해금 스킬 수 × 0.25%)
- 게임 중 달성한 해금은 현재 게임에 즉시 적용 안됨, 다음 게임부터
